## Linux Cmdlet Cheatsheet

If you're familiar with Linux and are needing to do some sysadmin work on Windows, it can be challenging remembering which cmdlet command will let you make a file, what it's aliases are and what it's counterpart is in Linux, which is why I've created the below list.

**Disclaimer:** All cmdlet commands shown are based on Windows 10 and all linux commands are based on the Debian 10.10.0. While a lot of these can be interchangable between Windows versions and linux distributions (distros), there can be some changes made that I'm unable to forsee.

Cmdlet | Description | Linux (Debian 10.10.0) Counterpart
-------|-------------|-----------------------------------
Get-ChildItem | Shows files in current directory (not recursive). | ls
